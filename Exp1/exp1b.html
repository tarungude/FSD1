Hyperlinks Demonstration

This program shows how to create hyperlinks using the <a> tag with href and target attributes.

1. Basic Hyperlink (href)

Visit Google using the href attribute.

2. Hyperlink with target="_blank"

Open Wikipedia in a new tab using target="_blank".

3. Hyperlink with target="_self"

Open GitHub in the same tab using target="_self".

4. Internal Hyperlink

Jump to the bottom of this page using href="#id".
